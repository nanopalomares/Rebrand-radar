# Rebrand Radar — Pitch Page

An interactive pitch page for the Rebrand Radar concept, tailored for Team Studio (Content and Discovery).

## What this is

A single-page pitch that covers:
- The problem and the data behind it (Gong, last 12 months)
- The rebrand lifecycle and where Canva's window sits
- The Radar concept and proof of concept results
- Why this matters specifically for Team Studio

## How to run it

No build step, no dependencies. Just open `index.html` in a browser.

```
open index.html
```

Or serve it locally:

```
npx serve .
```

## How to deploy

This is a single static HTML file. It can be hosted on:
- GitHub Pages (push to main, enable Pages in repo settings)
- Netlify (drag and drop the folder)
- Any static host

## Structure

```
rebrand-radar/
  index.html     — everything in one file
  README.md      — this file
```
